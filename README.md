# Exam Paper Repository

> A full-stack mobile app for browsing, searching, and downloading  
> past exam papers — built for students with low or no connectivity.

## Features

- **Offline access** — Download papers once, read them anywhere
- **Categorized search** — Filter by subject, year, and exam board
- **File downloads** — Save papers locally for offline study
- **Live in production** — Real users, actively maintained

## Tech Stack

| Layer    | Technology        |
|----------|-------------------|
| Frontend | Flutter (Dart)    |
| Backend  | Node.js / Express |
| Database | MySQL             |
| API      | REST              |

## Architecture

Flutter front-end → REST API (Node.js/Express) → MySQL  
Sole developer — designed, built, and deployed independently.
